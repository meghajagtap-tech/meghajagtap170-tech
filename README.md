# 🎭 Drama World - Acting Academy Portal

Drama World is a web application designed to manage an acting academy's curriculum and students. 
I am building this to master **Laravel** and **PHP** during my internship.

### 🛠️ Tech Stack
*   **Backend:** PHP 8.0 (Laravel Framework)
*   **Database:** MySQL
*   **Design:** Focused on a "realistic" and "settle" color palette.

### ✨ Key Features
*   **Course Management:** Handles modules like Method Acting and Audition Prep.
*   **Secure Auth:** User registration and login.
*   **Database Migrations:** Structured SQL schema.

### 🚀 How to Run
1. `composer install`
2. `php artisan migrate`
3. `php artisan serve`
